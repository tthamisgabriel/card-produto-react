# Card Produto - React

Aplicação simples desenvolvida em React utilizando Vite.

## Objetivo

Criar um componente de produto com:

- Nome do produto
- Preço
- Botão "Adicionar ao carrinho"

## Tecnologias

- React
- Vite
- Styled Components

## Funcionalidades

- Estilização utilizando CSS-in-JS (styled-components)
- Botão com mudança de cor dinâmica:
  - Cinza quando o produto não está adicionado
  - Verde quando o produto está adicionado

## Observação

A aplicação apresenta dois estados do botão para demonstrar a mudança visual com base na prop `adicionado`.

## Como executar

```bash
npm install
npm run dev
